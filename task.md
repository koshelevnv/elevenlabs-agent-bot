# Тестовое задание — Automation Developer

## О вакансии

Мы ищем Automation Developer на полную занятость с опытом работы в n8n, ElevenLabs и других платформах автоматизации.

**Требования:**
- Уверенный практический опыт работы с n8n
- Глубокое знание MySQL и SQL
- Опыт работы с REST API
- Дополнительный опыт разработки/программирования — значительное преимущество

---

## Тестовое задание

> Копирование этого задания в ИИ и отправка результата не пройдут проверку. Мы ищем разработчика, который умеет самостоятельно мыслить, правильно планировать, понимать требования и корректно их реализовывать.

### Цель

Создать Telegram-бота, который позволяет пользователям управлять своими голосовыми агентами ElevenLabs.

---

### Требования

#### 1. Структура базы данных

Создать MySQL-базу данных, которая связывает:
- ID аккаунтов пользователей Telegram
- Голосовых агентов ElevenLabs

База данных должна поддерживать несколько пользователей и несколько голосовых агентов.

#### 2. Функции Telegram-бота

Через бота пользователь должен иметь возможность:
- Изменять промпт голосового агента ElevenLabs
- Обновлять приветственное сообщение агента
- Обновлять базу знаний / контент агента

#### 3. Безопасность

Реализовать меры безопасности, гарантирующие:
- Пользователь может просматривать и изменять только своих агентов
- Пользователь не может видеть или изменять агентов других пользователей

#### 4. Выбор агента

Через меню Telegram-бота пользователь должен иметь возможность:
- Просматривать список доступных голосовых агентов
- Выбирать, каким агентом управлять

---

### Технические требования

- Использовать **n8n** для автоматизации воркфлоу
- Использовать **MySQL** для управления базой данных
- Использовать **REST/API** интеграции с ElevenLabs и Telegram
- Предпочтительна чистая и организованная структура воркфлоу

---

### Что нужно предоставить

- Экспортированный `.json` воркфлоу n8n
- Структуру MySQL (схему базы данных)
