# ElevenLabs Voice Agent Manager — Telegram Bot

Telegram-бот для управления голосовыми агентами ElevenLabs. Каждый пользователь привязывает свой API ключ и управляет только своими агентами.

## Стек
- **n8n** — воркфлоу автоматизации
- **MySQL** — хранение пользователей и агентов
- **Telegram Bot API** — интерфейс
- **ElevenLabs API** — управление агентами

## Архитектурное решение

Каждый пользователь хранит свой `elevenlabs_api_key` в БД. Все запросы к ElevenLabs выполняются с ключом конкретного пользователя — изоляция данных гарантирована на уровне API.

При каждом `/agents` список синхронизируется с ElevenLabs: новые агенты добавляются, удалённые убираются.

## Запуск

1. Выполнить `schema.sql`
2. Импортировать `workflow.json` в n8n
3. Настроить credential MySQL
4. Вставить Telegram Bot Token вместо `YOUR_TELEGRAM_BOT_TOKEN`
5. Активировать воркфлоу

## Использование

| Команда / действие | Результат |
|---|---|
| `/start` | Регистрация, запрос API ключа |
| Отправить ElevenLabs API ключ | Привязка аккаунта, показ списка агентов |
| `/agents` | Актуальный список агентов |
| Выбор агента | Меню: промпт, приветствие, база знаний |
